# Compiler_ChikiWiki
A custom compiler for the ChikiWiki programming language that performs lexical analysis, parsing, semantic validation, and code execution through an Abstract Syntax Tree (AST).
A custom compiler developed for the ChikiWiki programming language, designed to demonstrate the core phases of compiler construction. The project implements lexical analysis, syntax analysis, semantic checking, and Abstract Syntax Tree (AST) generation to process and validate source code. The compiler translates user-written programs into an executable representation while providing meaningful error detection and reporting.

## Features

* Lexical Analysis (Tokenization)
* Syntax Analysis (Parsing)
* Abstract Syntax Tree (AST) Construction
* Semantic Validation
* Error Detection and Reporting
* Custom Programming Language Support
* Modular Compiler Architecture

## Technologies

* C++
* Data Structures & Algorithms
* Compiler Design Concepts
* AST-Based Execution

## Learning Outcomes

This project demonstrates practical implementation of compiler design principles, including tokenization, parsing techniques, syntax validation, semantic analysis, and program execution through AST traversal.
